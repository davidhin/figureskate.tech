## Database
Running dumpDB.sh will dump the current database.

Install MySQL APT https://dev.mysql.com/downloads/repo/apt/
```
sudo dpkg -i /PATH/version-specific-package-name.deb
sudo apt-get update
sudo apt-get install mysql-server
```
